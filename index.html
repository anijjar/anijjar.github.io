Motor Controller

Custom PCB designed using Altium, Saturn Toolkit, and LTSpice · Fabricated by JLCPcb

This project was a requirement from AUVIC, where raw 24VDC needed to be distributed to eight Blue Robotics ESCs. Other requirements included temperature/pressure sensors, PWM output, CAN/USB/UART communication, current feedback, and togglability. My solution included a hardware selectable regulation, where the voltage can be brought down to 12/16/20 volts via jumpers, so that constant voltage can be provided to the speed controllers. A 4 layer stackup was used and the layers were stiched together to provide more copper for the current, also, the top soldermask was removed on the power traces so solder could be applied to decrease hot spots and increase current density. The current feedback can provide information to the on-board microcontroller, which can regulate the delivered current via a N-Channel MOSFET as needed.

N-Bit Kogge-Stone Adder

Built with Vivado and VHDL · Partner: Nic Richardson

The Kogge-Stone Adder is a digital arithmetic logic circuit that can perform addition operations with an increased performance by reducing fan-out compared to the Brent-Kung adder, with the cost of more area. Our implementation abstracts the bit size with VHDL, so any size adder can be synthesised, however it should be noted that synthesizing high bit count adders may not be feasible due to the large number of flip flops and wiring required.

Hermes - A BLDC Electronic Speed Controller running Field-Oriented Control

Capstone Project · Partners: Evan Maier and Avary Newell

As a graduation project, our team designed a PCB to drive a three-phase BLDC motor as well as wrote custom firmware. Field-Oriented Control is a variable frequency drive control technique for induction motors by regulating the stator fields to get the maximum torque, while minizing magnetism losses. This can be done on BLDC motors also. By using the rotor position, obtained via back-emf zero-point crossing, an encoder, or hall sensors, the field currents can be manipulated such that the rotor is spinning as smooth as possible. This can be further improved by using space vector PWM, a commutation technique that can drive half-bridge inverters such that the torque ripple is minimized. This project started with a thorough simulation on Simulink with the Motor Control Blockset to determine PI parameters and to check feasibility, then moved forward on hardware/firmware development.
