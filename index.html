Projects

Audio Effects

This board created audio effects using the fast fourier transform algorithm. The two types of effects implemented were the echoing and pitching effects. How it worked was it took an audio signal input from a 2.5mm AUX cable and then passed it through a butter-worth filter to remove any high frequency noise, before entering the ADC of a STM microcontroller. The effects can be enabled/disabled using two push buttons. Afterwards, the signal is outputted throigh the DAC into two filters before being played on a repurposed head-phone speaker.

CAN Dongle

Built while I was doing my co-op with the Autonomous Underwater Vehicle Interdisciplinary Club. This board converts between USB packets and CAN messages to allow the team to debug PCBs that use the CAN bus communication protocol. It display/sends messages through a terminal emulator, such as Tera Term.
