Street Traffic Simulator

Simualtes street traffic by using LEDs and shift-registers. The flow of traffic is adjusted by using a potentiometer, at which case the traffic light spends more time green than red.

Core Concepts: C | RTOS | Deadline-Driven Schedulars

Final Product

The electronics were placed on two breadboards and were controlled using a STM board; The system worked brillantly. 4 RTOS tasks were used: one to read the potentiometer, one to generate traffic, one to adjust traffic lights, and one to output the LED states.

STM CAN Bootloader

Allows firmware to be updated during run-time. Useful in remotely deployed machines

Current Progress

The bootloader is stored in flash memory at segment 1. Currently I have kept the allocated RAM size defaulted, but I will posibly reduce it down to a single page if it is taking little resources so the main app has more memory to work with.
