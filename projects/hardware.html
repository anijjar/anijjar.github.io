Pinger

Designed to test hydrophone systems on autonomous underwater vehicels. Sends acoustic signals at 25,30,35, and 40 kHz using an ultrasonic transducer. Uses a CMOS-inverter @9Vdc to drive the timer output from an STM chip before it is fed into the transducer. It monitors the external temperature, humidity, and pressure with 2 sensors to prevent catastrophic damage in case of water leakage.

Core Concepts: Mixed-signal design | I2C | USB | Timers

Project Progress: 30% - Adding USB charging function

Audio Effects

Creates an echo/pitching effect through button presess. The pitch level and modes are displayed using the 7-segment LEDs. The audio signal is filtered and passed through a STM where the effect is applied using FFT, until it is sent out the speaker.

Core Concepts: Analog Signal processing | PCB layout

Project Progress: 100% - Completed (Aug 2019)

Final Product: Effects were working correctly. LED display was mis-layered, so a different signaling scheme was used using the working LEDs. The board was tested using Lady Gaga-Paparazzi. The results were poor: The song was accompined by static, but the it slightly improved after shifting the connection. When connecting peripherals, we should always use proper connectors.
